Dimensions

Dimensions is a design for an augmented reality (AR) presentation platform. Currently, there are no available platforms that can be used to create AR presentations, though there may be primitive platforms to create apps. The design includes the user interface and the core functions of the platform. The platform can be used to create presentations with a multitude of purposes from displaying architectural or engineering concepts to presenting a new form of surgery. The uses are limitless.

Making the Unreal Real

AR technologies pride themselves on being “reality” and recreating or enhancing reality. However, if the elements rendered by these technologies are unrealistic, the renderings are violating the purpose they were created for. Developers often have to sacrifice certain aspects of renderings to either during the real time run to successfully create the rendering with the available processing power/computing resources, or during development to complete the project in time. Losing some of these aspects are inevitable, but, which aspects are more important and valuable when considering the realism of a rendering? This original work will aid developers and designers in the AR/VR sphere in prioritizing aspects and understanding what is necessary for renderings to be realistic.
